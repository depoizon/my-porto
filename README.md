## ✨ Tentang Project

**Depoizon Portfolio** adalah website portfolio personal/profesional dengan nuansa futuristik, premium, dan interaktif.  
Project ini dibuat menggunakan **HTML, CSS, dan JavaScript murni** tanpa framework, dengan fokus pada:

- tampilan modern dan elegan
- animasi yang halus
- robot interaktif sebagai identitas visual utama
- UI glassmorphism dengan warna biru transparan
- performa yang tetap ringan untuk penggunaan harian

Website ini merepresentasikan identitas digital **Depoizon / Nur Riski Firdaus** sebagai:

- Community Builder
- Software Engineer
- AI Explorer
- Tool Developer
- Digital Innovator

## 🚀 Fitur Utama

- 🤖 **Interactive Robot Assistant**  
  Robot mengikuti arah kursor atau sentuhan untuk membuat halaman terasa hidup.

- 🔥 **Angry Mode**  
  Robot dapat berubah ke tema merah ketika core ditekan beberapa kali.

- ⚡ **Laser Attack Effect**  
  Saat mode merah aktif, robot dapat menembakkan laser ke arah kursor dengan cooldown tertentu.

- 🧊 **Premium Glassmorphism UI**  
  Panel transparan, blur halus, glow lembut, dan nuansa digital modern.

- 🌌 **Futuristic Visual Atmosphere**  
  Background teknologi, garis digital, lighting lembut, dan tampilan yang terasa premium.

- 📊 **Animated Stats & Skills**  
  Counter statistik dan progress keahlian yang muncul secara halus.

- 🕒 **Smart Time Greeting**  
  Salam otomatis menyesuaikan waktu lokal.

- 🧭 **Smooth Section Navigation**  
  Navigasi antar section dibuat nyaman dan responsif.

- 🧳 **Travel / Explore Gallery**  
  Menampilkan dokumentasi perjalanan real-life dalam layout yang lebih rapi.

- 📬 **Clean Contact Section**  
  Kontak utama seperti email, GitHub, dan web mail ditampilkan dengan struktur yang sederhana dan profesional.

- 📱 **Responsive Layout**  
  Tetap nyaman dibuka di desktop maupun mobile.
